# 📈 Stock Financial Health Analyzer

A Streamlit-based financial dashboard for analyzing stock price trends and basic financial health indicators.  
This project allows users to select a company from a categorized dropdown list or manually enter a stock ticker, then review market performance, key financial metrics, and a simple rule-based financial health assessment.

---

## 1. Project Overview

The **Stock Financial Health Analyzer** is designed as an educational and exploratory tool for stock analysis. It combines:

- historical stock market data
- basic company financial indicators
- visual trend analysis
- simple financial health scoring

The application first attempts to retrieve live data from **Yahoo Finance** using the `yfinance` library.  
If online retrieval is unavailable, it can fall back to **local CSV files** for both stock prices and financial information.

This project is intended for:

- academic coursework
- financial data visualization practice
- beginner-friendly stock analysis
- Streamlit dashboard development

## 5. Installation and Setup

### Step 1: Clone the repository

    git clone https://github.com/your-username/stock-financial-health-analyzer.git
    cd stock-financial-health-analyzer

### Step 2: Create a virtual environment

#### Windows

    python -m venv .venv

#### Activate on Windows PowerShell

    .\.venv\Scripts\Activate.ps1

If PowerShell blocks activation, run:

    Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

### Step 3: Install dependencies

    python -m pip install --upgrade pip
    python -m pip install -r requirements.txt

If you do not have `requirements.txt`, install manually:

    python -m pip install streamlit yfinance pandas plotly

## 6. Running the Application

## 9. Financial Health Scoring Logic

The financial health score is based on four dimensions.

### 1. Liquidity
Measured using **Current Ratio**
- `>= 1.5` → strong
- `>= 1.0` → acceptable
- `< 1.0` → weak

### 2. Solvency
Measured using **Debt-to-Equity Ratio**
- `< 50` → strong
- `< 100` → acceptable
- `>= 100` → higher risk

### 3. Profitability
Measured using **Profit Margins**
- `> 20%` → strong
- `> 10%` → decent
- `<= 10%` → weak

### 4. Growth
Measured using **Revenue Growth**
- `> 15%` → strong
- `> 0%` → positive
- `<= 0%` → weak

Each dimension contributes up to **25 points**, for a total maximum score of **100**.

### Overall Rating
- `85–100` → Excellent
- `70–84` → Good
- `50–69` → Moderate
- `< 50` → Weak

> This scoring system is simplified and rule-based for educational use.

## 12. Limitations

Although useful as an educational dashboard, this project has several limitations:

- Yahoo Finance data may occasionally be incomplete or unavailable
- Some financial fields may return missing values depending on the ticker

- No advanced forecasting or machine learning is included
- No portfolio optimization or risk-adjusted return model is implemented
